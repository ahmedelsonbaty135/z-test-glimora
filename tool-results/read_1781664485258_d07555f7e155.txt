    10→١. مقدمة عامة
    11→تهدف هذه الوثيقة إلى تقديم مواصفات تفصيلية كاملة لمتجر إلكتروني فاخر متخصص في بيع المجوهرات الشخصية (أساور وقلائد بأسماء) مع إمكانية التخصيص. تم إعداد هذه الوثيقة بناءً على تحليل فيديو مرجعي لموقع مشابه (justuniques.shop) مع تطوير المميزات وإضافة عناصر الذكاء الاصطناعي.
    12→الوثيقة مقسمة إلى قسمين رئيسيين: القسم الأول يحتوي على المواصفات التفصيلية الكاملة للموقع (PRD) وهي موجهة للمطور أو الوكالة المنفذة. القسم الثاني يحتوي على Prompt جاهز للنسخ مباشرة إلى أي أداة ذكاء اصطناعي (Cursor, Claude, ChatGPT, v0, Lovable) لبناء الموقع خطوة بخطوة.
    13→الجمهور المستهدف من الموقع: عملاء من مصر بشكل أساسي، يبحثون عن مجوهرات شخصية مخصصة كهدية أو للاستخدام الشخصي. السعر متوسط إلى مرتفع، والتركيز على الجودة والتخصيص والخدمة المميزة عبر WhatsApp.
    14→٢. نظرة عامة على المشروع
    15→٢.٠ هوية البراند — GLIMOKA
    16→البراند اسمه GLIMOKA، ولوجو بيميزه لون خمري فاخر (Burgundy #6A1B35) كخلفية، مع نص أبيض نظيف وخط sans-serif حديث، ولمسات عضوية من خطوط منحنية ناعمة بلون خمري فاتح (#8B3A4F). الديزاين العام للوجو minimalist luxury مع إحساس بالدفء والنعومة. اسم البراند مكتوب بحروف لاتينية كبيرة (uppercase) مع letter-spacing عالي، وفيه لمسة سكاندينافية من الحرف Å المخصص.
    17→الشخصية اللي بيبثها البراند: فخامة معاصرة، دفء، أناقة، قرب من العميل (مش برود الفخامة الكلاسيكية). الموقع لازم يعكس نفس الشخصية — خمري وcreamي وذهبي وردي، مع مساحات بيضاء كتير، وصور احترافية، وأنيميشن سلس. اسم GLIMOKA لازم يظهر بوضوح في الهيدر بنفس ستايل اللوجو.
    18→٢.١ فكرة المشروع
    19→متجر إلكتروني فاخر لبيع المجوهرات الشخصية المخصصة (أساور، قلائد، خواتم) بحيث يستطيع العميل إدخال اسمه أو اسم شخص محب ويتم تصنيع القطعة خصيصاً له. الموقع يقدم تجربة مستخدم راقية مع إمكانية معاينة المنتج بشكل تفاعلي، والتواصل المباشر مع خدمة العملاء عبر WhatsApp لإتمام الطلب أو الاستفسار.

    29→٣. المواصفات التقنية (Tech Stack)
    30→الموقع مبني بالكامل على Next.js 14 مع TypeScript لبناء كود آمن وقابل للصيانة. الواجهة الأمامية تستخدم Tailwind CSS للتصميم السريع والمتسق. قاعدة البيانات PostgreSQL مع Prisma ORM. المصادقة عبر NextAuth.js. الـ State management عبر Zustand. الدفع عند الاستلام فقط (لا حاجة لبوابة دفع إلكترونية في المرحلة الأولى).

   100→٣.٢ البنية المعمارية للمشروع
   101→المشروع يتبع بنية Monorepo بسيطة مع فصل واضح بين الـ App Router pages (للواجهة الأمامية) و API routes (للـ backend). قاعدة البيانات على Vercel Postgres أو Supabase. التخزين للصور على Vercel Blob أو Cloudflare R2. الـ Admin dashboard يكون جزء من نفس المشروع تحت مسار /admin مع حماية إضافية.
   102→هيكل المجلدات المقترح:
   103→◆  app/ (للواجهة الأمامية - App Router)
   104→◆  app/(shop)/ (صفحات المتجر: home, products, product/[slug], cart, checkout)
   105→◆  app/(auth)/ (login, register, forgot-password)
   106→◆  app/(account)/ (profile, orders, wishlist, addresses)
   107→◆  app/admin/ (لوحة التحكم الإدارية)
   108→◆  app/api/ (API routes للـ backend)
   109→◆  components/ (مكونات React قابلة لإعادة الاستخدام)
   110→◆  lib/ (utilities, prisma client, auth config)
   111→◆  prisma/ (schema.prisma + migrations)
   112→◆  public/ (الصور والأيقونات الثابتة)
   113→◆  types/ (TypeScript types عامة)
   114→٤. الصفحات الرئيسية للموقع
   115→الموقع يحتوي على مجموعة صفحات مصممة بعناية لتوفير تجربة تسوق متكاملة. كل صفحة محسّنة لسرعة التحميل (LCP < 2.5s) وSEO (meta tags, structured data, sitemap). التصميم Mobile-first مع دعم كامل للـ Desktop.
   116→٤.١ الصفحة الرئيسية (Home Page) — /
   117→الصفحة الرئيسية هي واجهة المتجر وأهم صفحة للانطباع الأول. تحتوي على Hero section بصورة عالية الجودة لمجموعة مجوهرات مع نص ترحيبي وزر CTA رئيسي. أسفلها قسم المنتجات المميزة (Featured) وقسم الأكثر مبيعاً (Best Sellers) وقسم العروض الحالية. ثم قسم About قصير介绍 عن البراند وقسم الشهادات (Testimonials) وفي الآخر Newsletter signup وروابط التواصل الاجتماعي.
   118→عناصر الصفحة الرئيسية:
   119→1. Hero Section: صورة فاخرة + عنوان رئيسي + زر تسوق الآن + countdown timer للعروض
   120→2. Featured Categories: 3-4 فئات (أساور، قلائد، خواتم، عروض) بصور دائرية
   121→3. Best Sellers: عرض 4-8 منتجات الأكثر مبيعاً مع تقييمات وخصومات
   122→4. Personalization Highlight: قسم يشرح كيفية تخصيص المنتج بفيديو أو GIF
   123→5. Testimonials: شهادات عملاء حقيقيين مع اسم وصورة ونجوم تقييم
   124→6. About Brand: قصة البراند مع صورة المؤسس/المحل
   125→7. Instagram Feed: عرض آخر منشورات Instagram (via API)
   126→8. WhatsApp CTA: زر عائم أخضر في كل الصفحات للتواصل المباشر
   127→9. Newsletter: اشتراك في النشرة البريدية للحصول على خصم 10%
   128→10. Footer: روابط، سياسات، طرق دفع، وسائل تواصل
   129→٤.٢ صفحة كل المنتجات (Products Page) — /products

   132→◆  Grid View متجاوب (2-4 columns حسب حجم الشاشة)

   146→2. Product Title: اسم المنتج بخط كبير وبارز
   147→3. Rating Summary: متوسط التقييم + عدد المراجعات + روابط للمراجعات

   152→8. Quantity Selector: - / + buttons مع حد أدنى وأعلى
   153→9. Add to Cart Button: زر كبير أسود مع أيقونة سلة
